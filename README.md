# 功能备注

H5应用需要一个app壳。 
壳APP框架功能需求： 
1、 根据配置文件切换H5应用链接，配置文件可以在线更新并本地保存； 
2、 支持拍照、扫码、录音接口；✔️ 
3、 支持微信＼微博分享接口； 
4、 尽可能适配低版本系统（考虑使用人员有许多老手机），屏幕大小自适应。 
5、 提供APP升级功能。 
6、 获取设备信息接口：手机网络状态＼手机系统信息＼位置信息＼剪贴板＼系统文件\通讯录 
7、 拨打电话＼发送短信息  ✔️
8、 选择文件以供上传
